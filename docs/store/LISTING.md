> 근거는 `docs/COMPETITIVE_POSITIONING.md`. 과장·거짓 금지, 아동 대상 결제 유도 문구 금지.

## 앱 이름 (최대 30자)
```
딱풀: 초등 사고력 수학
```

## 짧은 설명 (최대 80자)
```
학년 대신 난이도 1~10, 딱 맞는 사고력 문제. 인터넷·광고·수집 없이 기기 안에서.
```

## 자세한 설명 (최대 4000자)
```
딱풀은 '학원 사고력'을 태블릿에 담되, 학원의 부담은 뺐습니다.

■ 계산 드릴이 아니라, 진짜 '생각하는' 문제
연습장을 펴고 5분 고민하는 다단계 추론·전략·퍼즐 문제만 담았습니다. 단순 계산 반복이 아니라, 왜 그런지 스스로 발견하며 '생각하는 힘'을 기릅니다.

■ 학년이 아니라 난이도 1~10으로
같은 나이라도 실력은 다릅니다. 딱풀은 학년을 없애고 난이도 1~10으로만 나눠, 아이가 문제를 풀 때마다 실력에 '딱 맞는' 다음 문제를 자동으로 골라 줍니다. 최상위 단계는 경시·올림피아드급까지 이어집니다.

■ 모든 문제에 단계별 풀이 + 심화 풀이
모든 문제에 '왜 그렇게 푸는지' 단계별 풀이가 들어 있어 아이 혼자서도 이해합니다. 게다가 '이 유형이 왜 그런지, 다른 문제로 어떻게 이어지는지'까지 짚는 심화 풀이로 한 걸음 더 깊이 들어갑니다. 틀린 문제는 '어디서 빠졌는지'를 짚어 오답이 실력이 되게 합니다.

■ 부모를 위한 리포트 — 통계를 넘어 '다음 한 걸음'
아이가 어느 영역이 강하고 어디가 약한지 한눈에 보이는 리포트를 제공합니다. 숫자만 보여주는 데서 멈추지 않고, "지금은 이 부분을 함께 보세요" 같은 실행 가능한 코칭까지 드립니다. 리포트는 PDF로 저장·인쇄해 보관할 수 있습니다.

■ 계정도, 광고도, 자동결제도 없습니다
문제 풀이·채점·추천·리포트 같은 핵심 학습은 인터넷 없이 기기 안에서 완결됩니다(읽어주기 음성과 해설 영상만 원할 때 내려받아요). 회원가입이 필요 없고, 아이의 학습 기록은 단 한 건도 기기 밖으로 나가지 않습니다(아동 데이터 수집 0). 광고와 결제 유도는 아이에게 보여주지 않습니다.

■ 별사탕·기프티콘으로 낚지 않습니다
코인·리더보드·실물 보상 같은 자극 대신, '푸는 재미'와 '해냈다'는 경험으로만 학습을 이어갑니다. 정답보다 '어떻게 풀었는지'를 칭찬하도록 설계했습니다.

■ 지금은 출시 기념, 전부 무료
2026년 10월 31일까지 모든 난이도·심화 풀이·리포트를 무료로 쓸 수 있습니다. 결제도, 구독도, 자동 갱신도 없습니다.

- 대상: 초등 사고력 수학(수와 연산 / 변화와 관계 / 도형과 측정 / 자료와 가능성)
- 사용자: 아이(문제 풀이) / 부모(리포트)
- 태블릿·휴대폰 모두 지원

생각하는 힘은 하루 한 문제에서 시작됩니다. 딱풀과 함께해 보세요.
```

## 태그라인 후보 (프로모/맘카페)
- 학원 사고력, 워크북 값에 — 적응형인데 사고력.
- 인터넷 없이, 광고 없이, 미끼 없이. 아이 실력에 딱.
- 구독 아니라 이용권. 아이 데이터 0, 자동결제 0.
