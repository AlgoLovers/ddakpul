## 앱 이름 (최대 30자)
```
딱풀: 초등 사고력 수학
```

## 짧은 설명 (최대 80자 · 앞 26자에 승부)
```
5분 생각하는 사고력 수학. 실력에 맞춰 다음 문제 자동 추천, 단계별 풀이와 학습 리포트까지.
```
후보(A/B용):
- `계산 드릴 대신 퍼즐·추론 사고력 문제. 무료·광고 없음·오프라인, 유아부터 성인까지.`
- `적응형 사고력 수학. 실력 따라 딱 맞는 문제 추천, 부모용 학습 리포트 제공.`

## 자세한 설명 (최대 4000자)
```
딱풀은 유아부터 성인까지, 연습장을 펴고 5분 생각하는 사고력 수학 앱입니다. 계산 드릴이 아니라 퍼즐·추론·전략 문제를 풀며, 정답 여부와 실력에 따라 다음 문제를 자동으로 추천합니다.
✅ 완전 무료 · 광고 없음 · 인터넷 없이도 학습 · 아동 데이터 수집 0

■ 계산 드릴이 아니라, 진짜 '생각하는' 문제
연습장을 펴고 5분 고민하는 다단계 추론·전략·퍼즐 문제만 담았습니다. 단순 계산 반복이 아니라, 왜 그런지 스스로 발견하며 '생각하는 힘'을 기릅니다.

■ 학년이 아니라 난이도 1~10으로 — 실력에 딱 맞게
같은 나이라도 실력은 다릅니다. 딱풀은 학년을 없애고 난이도 1~10으로만 나눠, 아이가 문제를 풀 때마다 실력에 '딱 맞는' 다음 문제를 자동으로 골라 줍니다. 최상위 단계는 경시·올림피아드급까지 이어집니다.

■ 모든 문제에 단계별 풀이 + 심화 풀이
모든 문제에 '왜 그렇게 푸는지' 단계별 풀이가 들어 있어 아이 혼자서도 이해합니다. '이 유형이 왜 그런지, 다른 문제로 어떻게 이어지는지'까지 짚는 심화 풀이로 한 걸음 더 깊이 들어갑니다. 틀린 문제는 '어디서 빠졌는지'를 짚어 오답이 실력이 되게 합니다.

■ 부모를 위한 리포트 — 통계를 넘어 '다음 한 걸음'
아이가 어느 영역(수와 연산·변화와 관계·도형과 측정·자료와 가능성)이 강하고 어디가 약한지 한눈에 보이는 리포트를 제공합니다. 숫자만 보여주는 데서 멈추지 않고, "지금은 이 부분을 함께 보세요" 같은 실행 가능한 코칭까지 드립니다. 리포트는 인쇄해 보관할 수 있습니다.

■ 계정도, 광고도 없이 — 안심하고 맡기는 학습
문제 풀이·채점·추천·리포트 같은 핵심 학습은 인터넷 없이 기기 안에서 완결됩니다(읽어주기 음성과 해설 영상만 원할 때 내려받아요). 회원가입이 필요 없고, 아이의 학습 기록은 단 한 건도 기기 밖으로 나가지 않습니다. 광고는 아이에게 보여주지 않습니다.

■ 별사탕·기프티콘으로 낚지 않습니다
코인·리더보드·실물 보상 같은 자극 대신, '푸는 재미'와 '해냈다'는 경험으로만 학습을 이어갑니다. 정답보다 '어떻게 풀었는지'를 칭찬하도록 설계했습니다.

- 대상: 사고력 수학(수와 연산 / 변화와 관계 / 도형과 측정 / 자료와 가능성)
- 사용자: 아이(문제 풀이) · 부모(리포트)
- 태블릿·휴대폰 모두 지원 · 완전 무료 · 광고 없음

생각하는 힘은 하루 한 문제에서 시작됩니다. 딱풀과 함께해 보세요.
```

## English listing

**App name (≤30)**
```
DdakPul: Thinking Math for Kids
```

**Short description (≤80)**
```
5-minute thinking-math. Adaptive problem picks, step-by-step solutions & reports.
```

**Full description**
```
DdakPul is a thinking-math app for ages 4 to adult — the kind of problem you open a scratch pad and think about for five minutes. Not calculation drills, but puzzles, reasoning and strategy, with the next problem chosen automatically from how you answer.
✅ Completely free · No ads · Works offline · Zero data collected from children

■ Real thinking, not drills
Only multi-step reasoning, strategy and puzzle problems — the kind worth five minutes of thought. Children discover *why* an answer works and build genuine thinking skills, not rote repetition.

■ By skill (levels 1–10), not by grade
Same age, different skill. DdakPul drops grade levels entirely and sorts by difficulty 1–10, picking the next problem that fits your child exactly. The top levels reach competition and olympiad difficulty.

■ Step-by-step + deeper solutions on every problem
Every problem explains *how* to solve it, so children understand on their own. A deeper explanation shows *why* the type works and how it connects to other problems. Wrong answers point to *where* the reasoning slipped.

■ A parent report that goes beyond numbers
See at a glance which of the four areas your child is strong or weak in — and get an actionable "next step," not just statistics. Reports can be printed.

■ No account, no ads — learning you can trust
Solving, grading, recommendation and reports all run on-device, fully offline (only optional read-aloud voices and solution videos are downloaded). No sign-up, and your child's records never leave the device. No ads shown to children.

■ No candy, no gift cards
No coins, leaderboards or physical rewards — just the joy of solving and the "I did it" moment. Designed to praise *how* you solved, not just the answer.

Thinking starts with one problem a day. Try DdakPul.
```

## 스크린샷 캡션 (마케팅 세트 · 동사+혜택 공식)
> 렌더링 자산: `docs/store/marketing/` (생성기 `gen_store_assets.py`). 첫 3장이 전환 결정 → 훅·차별점·부모 순.
1. **5분 생각하는 사고력 문제** — 계산 드릴이 아니라, 연습장 펴고 푸는 문제 (도형 넓이 문제 화면)
2. **실력 따라 다음 문제 자동 추천** — 단계별·심화 풀이로 이해까지 (정답+풀이 화면)
3. **부모를 위한 성장 리포트** — '다음 한 걸음'까지 알려주는 코칭 (Lv 리포트 화면)
4. **네 영역, 강점·약점 한눈에** — 어디가 강하고 어디를 더 볼지 (영역별 성취 화면)
5. **믿고 맡기는 학습** — 완전 무료·광고 없음·오프라인·아동 데이터 수집 0 (신뢰 슬라이드)
