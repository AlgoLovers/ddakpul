# 딱풀(DdakPul) 개인정보처리방침

딱풀은 이용자의 **어떤 개인정보도 수집하지 않습니다.**

- 이름, 생년월일, 연락처 등 개인정보를 묻거나 저장하지 않습니다.
- 계정·로그인이 없습니다.
- 모든 학습 기록은 이용자의 기기 안에만 저장되며, 외부 서버로 전송되지 않습니다(서버가 없습니다).
- 광고 및 제3자 분석(애널리틱스) 도구를 사용하지 않습니다.
- 핵심 기능(문제 풀이·채점·추천·리포트)은 인터넷 연결 없이 동작합니다.
  인터넷은 이용자가 원할 때 고품질 읽어주기 음성 모델을 내려받는 용도로만 사용됩니다.
- 만 14세 미만 아동의 개인정보를 수집하지 않습니다(개인정보보호법 준수).

앱 안의 모든 데이터는 이용자가 직접 삭제할 수 있습니다: **설정 → 학습 기록 초기화**,
또는 앱 삭제 시 전부 제거됩니다.

# DdakPul Privacy Policy (English)

DdakPul collects **no personal information whatsoever.**

- We never ask for or store names, birthdates, or contact details.
- There are no accounts and no sign-in.
- All learning records stay on the user's device and are never transmitted anywhere (there is no server).
- No ads. No third-party analytics.
- Core features work fully offline. The internet is used only when the user chooses to
  download an optional high-quality text-to-speech voice model.
- We do not collect personal information from children.

Users can delete all data at any time: **Settings → Reset learning history**, or by uninstalling the app.
