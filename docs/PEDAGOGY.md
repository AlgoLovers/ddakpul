# 딱풀 교육학 근거 (PEDAGOGY)

> 딱풀의 설계 결정이 어떤 연구에 기대고 있는지 기록한다. 문구·수치·알고리즘을 바꿀 때는
> 이 문서의 근거와 충돌하지 않는지 먼저 확인한다. (2026-07 리서치 기준)

## 1. 추천·난이도 설계

| 결정 | 근거 |
|---|---|
| 연습 정답률 70~85% 밴드 유지 (연속 2정답 ↑ / 2오답 ↓) | Rosenshine 수업 원리 #7 — 안내된 연습 성공률 ~80%가 최적. 그보다 낮으면 오류를 연습하고, 높으면 도전이 없다 |
| 정체 감지 → 해설 + 선수 개념 복귀 (`STAGNATION_WRONG=3`, 최근 10문항 창) | Beck & Gong "wheel-spinning": 10회 연습에도 3연속 정답 미달이면 문제를 더 줘도 해결 안 됨. 학생-스킬 쌍의 ~25%에서 발생 |
| 숙달 학습 + 즉각 피드백 자체의 정당성 | Hattie 종합: 피드백 d≈0.70, 숙달 학습 d=0.57 (평균 힌지 0.40 상회) |
| 그룹(유형) 인터리빙 — 학습지 출제에서 같은 유형 연속 금지 | Rohrer et al. 2020 RCT: 혼합 배치가 블록 연습 대비 1개월 후 61% vs 38%, d=0.83 |
| 문제 풀기 전 힌트·정답 노출 금지 | 인출 연습 효과 g=0.61 (Adesope 2017) — 스스로 답을 내는 시도 자체가 학습 |

## 2. 피드백 설계

| 결정 | 근거 |
|---|---|
| 채점 즉시, 정오답 + 정답 + 오개념 지목의 3단 피드백 | Van der Kleij 2015 메타분석: 설명형 피드백 ES=0.49 vs 정오답만 0.05. 수학에서 효과 최대. Fyfe & Rittle-Johnson: 초등은 즉각 피드백 > 지연 피드백 |
| 흔한 오답(보기)마다 오개념 1:1 매핑 (`commonMistakes`) | 오개념 기반 교정 피드백이 정답 제시보다 효과적. 4학년 대표 오개념: 분모끼리 더함, 받아올림 누락, 자릿값 혼동, 각도기 눈금 혼동 등 |
| 해설에 수직선·그림 표상 우선 (콘텐츠 확충 시) | WWC 2021 수학 개입 가이드 권고 ③④ — 반구체적 표상과 수직선 |

## 3. 동기 설계

| 결정 | 근거 |
|---|---|
| 과정 칭찬만 사용 — 전략·시도·개선을 짚는 문구 풀 (`praise_*` 배열) | Mueller & Dweck 1998: 지능 칭찬은 실패 후 끈기·성적 하락, 노력 칭찬은 반대. Gunderson 2013/2018: 부모의 과정 칭찬이 5년 뒤 성취까지 예측 |
| 오답 = "뇌가 크는 신호" 프레이밍, 부정 효과음·감점 없음 | Moser 2011 / Schroder 2017: 성장 마인드셋은 오류 후 정확도 상승. Kapur: 생산적 실패 |
| 하루 10문항 근접 목표 (`DAILY_GOAL_PROBLEMS=10`) | Bandura & Schunk 1981: 근접 목표가 실력·자기효능감·내적 흥미 모두 상승. 세션 적정량 10~15문항, 10~20분 |
| 관대한 스트릭 — 어제까지 이어졌으면 유지, 복귀 시 이어짐 | Duolingo 스트릭 연구: 손실회피는 강력하나 냉혹한 리셋은 이탈 트리거. 완충 장치 필수 |
| 정답 조건부 물질 보상(코인 등) 금지 | Deci, Koestner & Ryan 1999 (128개 실험): 예상된 조건부 보상은 내적 동기 훼손(과잉정당화). 언어적·정보적 피드백은 동기 상승 |
| 리더보드·또래 비교 없음 | K-12 게이미피케이션 메타분석: 초등에서 역효과 위험. 진행바·피드백 우선, 배지는 소수, 리더보드 배제 |

## 4. 학부모 리포트 설계

| 결정 | 근거 |
|---|---|
| 문장형 인사이트를 차트보다 먼저 | 학부모 대시보드 연구: 부모가 원하는 건 해석이 끝난 문장과 "다음 행동" |
| 지표 우선순위: 개념 숙달 → 취약 개념+행동 제안 → 정답률 추이 → 학습 습관(스트릭) → 오답 해소율 | Wizidoo/Pandai 등 대시보드 사례 연구 종합 |
| 숙달을 점수가 아닌 단계 라벨로 (숙달/연습 중/보강 필요) | Khan Academy: 단계가 점수보다 직관적. IXL SmartScore의 "성적화" 부작용 반면교사 |
| 부모 팁 5종 고정 노출 (칭찬법·수학 불안 전이 경고 포함) | Maloney & Beilock 2015: 수학 불안 부모가 숙제를 자주 도울수록 자녀 성취 하락. Berkowitz 2015: 구조화된 짧은 수학 대화는 효과적 |
| 또래 비교·석차 없음 | 수행목표 자극 → 학습목표 밀어냄 + 부모 과잉 개입 유발 |

## 5. 인쇄 학습지 설계

| 결정 | 근거 |
|---|---|
| A4당 문장제 4~6문항 상당(풀이 공간 포함), 총 5/10/15 옵션 | 3P Learning·워크시트 설계 가이드: 25문항 이상이면 완료율 40% 급락. 구몬: 하루 1~2장, 10~15분 |
| 풀이 공간을 문항 바로 아래 넉넉히 | 여백이 넉넉하면 "쉬워 보여서" 시작·완료율 상승 |
| 정답·해설은 별지 분리, 해설 1~2줄 포함 | 구몬·눈높이 관례 — 학부모 채점용 |
| 쉬움→어려움 순 배치 + 같은 유형 연속 금지 | 구몬 스몰스텝(성취감 반복) × Rohrer 인터리빙의 절충 |

## 6. 다음 개선 후보 (리서치가 제안, 아직 미구현)

1. **개념별 Elo-lite** — 학생 θ/문항 d 레이팅, 기대정답률 60~75% 문항 출제. 온디바이스 계산 가능 (Pelánek)
2. **BKT-lite 숙달 판정** — 고정 파라미터(L0=0.2, T=0.15, S=0.1, G=0.25) 베이즈 갱신, P(안다)>0.95면 출제 중단. "연속 N정답"보다 실수·찍기에 강건
3. **Leitner 박스 복습** — 숙달 개념을 1·3·7·14·30일 간격 재출제, 세션 배합 워밍업 10% + 신규 60% + 복습 30%
4. **오답 직후 쌍둥이 문항 재출제** — 교정 후 즉시 성공 경험 (매쓰플랫 방식)
5. **세션 소프트 컷** — 15~20분 또는 목표 도달 시 "오늘은 여기까지" (후반 피로 오답이 난이도 하락 오판을 만들기 전에)
6. **유창성 블록** — 기초 연산 한정 1~2분 시간제한 활동, "지난번의 나" 대비 표시 (WWC 권고 ⑥)
